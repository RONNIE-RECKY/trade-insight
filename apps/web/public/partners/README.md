# Partner logos

Drop the partner logo image files here. The footer renders them automatically;
until a file exists, the footer falls back to showing the partner's name as text.

Required files (exact names):

- `hive-konnect.png`  — the HIVE Konnect bee logo
- `broke-boyz.png`    — the Broke Boyz logo

Tips for a clean, professional look:
- **Transparent PNG** is best (no background box). Use a free tool like remove.bg
  to strip the background, then export as PNG.
- Logos display on a white chip in the footer, so a logo with dark text/graphics
  reads well. Aim for roughly 300–600px wide, transparent background.
